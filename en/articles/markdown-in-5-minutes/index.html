Markdown in 5 minutes

March 28, 2026

Learning

Every article on this blog is written in Markdown. It's a simple text format that turns into HTML. Here are the basics.

The essential syntax

I want… I write…
Bold **text**
Italics *text*
Heading # Heading
Link [text](url)
Code `code`

Lists

A list with dashes:

  • First item
  • Second item
  • Third item

Code blocks

To show code you use three backticks:

console.log("Hello, world");

And that's about it. With this you can already write 90% of an article without touching any HTML.
