Why I chose 11ty

May 20, 2026

Development Learning

When I started this site I had to pick a tool to build it. I ended up choosing 11ty (Eleventy) and here's why.

What a static site generator is

A static site generator takes simple files (Markdown, templates) and produces plain HTML. There's no database and no server running code on each visit: the output is just files ready to serve.

The reasons

  • Simplicity: I write in Markdown and 11ty does the rest.
  • Speed: being static HTML, the site loads very fast.
  • No lock-in: the output is regular HTML, so I can host it anywhere.

I didn't need anything complex. I needed something that stayed out of the way.

For a personal site with articles, it was exactly what I was looking for.
