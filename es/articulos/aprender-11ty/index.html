Por qué elegí 11ty

20 de mayo de 2026

Cuando empecé este sitio tenía que elegir una herramienta para generarlo. Acabé eligiendo 11ty (Eleventy) y aquí explico por qué.

Qué es un generador de sitios estáticos

Un generador de sitios estáticos toma archivos sencillos (Markdown, plantillas) y produce HTML puro. No hay base de datos ni servidor ejecutando código en cada visita: el resultado son ficheros listos para servir.

Las razones

  • Sencillez: escribo en Markdown y 11ty hace el resto.
  • Rapidez: al ser HTML estático, el sitio carga muy rápido.
  • Sin ataduras: el resultado es HTML normal, lo puedo alojar en cualquier sitio.

No necesitaba algo complejo. Necesitaba algo que no se interpusiera en el camino.

Para una página personal con artículos, era justo lo que buscaba.
