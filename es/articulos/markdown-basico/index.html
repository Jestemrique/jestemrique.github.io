Markdown en 5 minutos

28 de marzo de 2026

Todos los artículos de este blog están escritos en Markdown. Es un formato de texto sencillo que se convierte en HTML. Aquí va lo básico.

La sintaxis esencial

Quiero… Escribo…
Negrita **texto**
Cursiva *texto*
Título # Título
Enlace [texto](url)
Código `código`

Listas

Una lista con guiones:

  • Primer punto
  • Segundo punto
  • Tercer punto

Bloques de código

Para mostrar código se usan tres acentos graves:

console.log("Hola, mundo");

Y poco más. Con esto ya se puede escribir el 90 % de un artículo sin tocar nada de HTML.
